# Nome del prodotto/pacchetto: Query Studio for Marketing Cloud

## Breve descrizione sintetica
Salesforce Labs tool for quickly running SQL queries in Marketing Cloud.

## Oggetti principali
- Marketing Cloud data extensions, subscribers and synchronized data sources.
- Query activities and result data extensions.
- Business unit users and permissions.

## Funzionalita principali
- Interactive SQL query execution.
- Quick data inspection for Marketing Cloud.
- Query prototyping before automation.

## Configurazioni principali
- Package access in Marketing Cloud.
- Business unit permissions and data extension visibility.
- Query result handling and data retention.

## Best practice
- Use for investigation and prototyping, not as unmanaged production automation.
- Avoid querying sensitive data unless access and retention are approved.
- Move production logic into governed automations.
- Validate query cost and data volume before repeated use.
