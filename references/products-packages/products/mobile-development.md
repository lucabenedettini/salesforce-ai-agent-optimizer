# Nome del prodotto/pacchetto: Mobile Development

## Breve descrizione sintetica
Development and configuration for Salesforce mobile experiences, mobile-ready LWC, offline behavior and custom mobile apps.

## Oggetti principali
- LightningComponentBundle, FlexiPage, CustomApplication, NavigationMenu, CompactLayout.
- QuickAction, GlobalValueSet, PermissionSet, Mobile settings metadata.
- Offline data sets, Briefcase Builder configuration, Mobile SDK app artifacts where used.

## Funzionalita principali
- Salesforce mobile app customization.
- Mobile-ready Lightning Web Components.
- Offline access and sync for supported mobile use cases.
- Custom mobile apps with Mobile SDK when standard app is not enough.

## Configurazioni principali
- Lightning app navigation, mobile cards, actions and compact layouts.
- FlexiPage activation by app, record type and form factor.
- Offline settings, briefcases, data sync and mobile security.
- Device feature access, responsive UI and performance constraints.

## Best practice
- Design for mobile form factor, network latency and offline failure first.
- Use mobile-ready LWC patterns and avoid desktop-only UI assumptions.
- Validate object/field access, offline data scope and conflict behavior.
- Test on real devices or mobile previews before release.

## Contesto versione
Prima della pianificazione leggere `references/salesforce-current-version.md`. Verificare la release della target org e la disponibilita' effettiva delle funzionalita'; per prodotti distribuiti anche come managed package o add-on, verificare la versione installata nella org quando rilevante.
