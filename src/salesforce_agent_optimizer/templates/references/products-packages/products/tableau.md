# Nome del prodotto/pacchetto: Tableau

## Breve descrizione sintetica
Analytics Salesforce per esplorazione dati, dashboard, reporting visuale, embedded analytics e insight decisionali.

## Oggetti principali
- Tableau workbook, data source, dashboard, project, site, user/group.
- Salesforce reports/datasets, CRM Analytics assets where integrated.
- Connected app, permission set, external credential and embedded analytics metadata.

## Funzionalita principali
- Data visualization, dashboards and self-service analytics.
- Embedded analytics in Salesforce.
- Data connections to CRM, Data 360 and external systems.

## Configurazioni principali
- Sites, projects, workbooks, data sources, refresh schedules.
- User/group access, row-level security and connected apps.
- Embedded component configuration and single sign-on.
- Data extracts, live connections and governance.

## Best practice
- Define source of truth and refresh cadence before dashboard design.
- Align Salesforce sharing with Tableau row-level security.
- Keep calculations documented and avoid duplicating business metrics.
- Validate performance with realistic data volume.

## Contesto versione
Prima della pianificazione leggere `references/salesforce-current-version.md`. Verificare la release della target org e la disponibilita' effettiva delle funzionalita'; per prodotti distribuiti anche come managed package o add-on, verificare la versione installata nella org quando rilevante.
